\Molajo\IoCServiceProviderAdapter

Service Provider Adapter

Driven by the Service Provider Controller to interact with the injected Service Provider in order to resolve dependencies by constructing dependent classes and construct the primary class, itself, once all dependencies are available The constructed class will be stored within the IoC Container if so specified in the Service Provider options.
